# 赶早八 · Campus Rush

大学校园主题跑酷游戏，Python + Pygame 实现。

## 运行

```bash
uv sync
uv run python src/main.py
```

## 操作

| 按键 | 动作 |
|------|------|
| 空格 / ↑ / W | 跳跃（可二段跳） |
| ↓ / S | 下蹲滑铲（躲避上方障碍物） |
| 鼠标点击 | 上半屏跳跃 / 下半屏滑铲 |

## 玩法

大学生赶早八课，在校园道路上奔跑，躲避地面和上方的各种障碍物。速度随分数递增，障碍物种类和密度逐渐增加。

### 障碍物

| 类型 | 障碍 | 应对 |
|------|------|------|
| 🟫 地面 | 垃圾桶 | 跳跃 |
| 🟫 地面 | 校园指示牌 | 二段跳 |
| 🟫 地面 | 警戒线路障 | 跳跃（较宽） |
| 🟫 地面 | 食堂餐车把手 | 跳跃或滑铲 |
| 🟫 地面 | 减速带 | 必须跳跃 |
| 🟫 地面 | 电动车 | 跳跃或滑铲均可 |
| 🔴 上方 | 低垂树枝 | 下蹲滑铲 |
| 🔴 上方 | 校园横幅 | 下蹲滑铲 |
| 🔴 上方 | 道闸横杆 | 下蹲滑铲 |

### 分数机制

- 每 400 分速度提升一级
- 300 分后解锁全部障碍物类型
- 800 分后高难度障碍物出现概率增加
- 最高分自动保存

## 技术栈

- Python 3.12+
- pygame-ce 2.5+
- uv 包管理
